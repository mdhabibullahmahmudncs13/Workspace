Project Progress

Project Overview

This project aims to create an interactive dashboard that visualizes climate change data from NASA's Earth Science datasets. The visualization will help researchers and the general public understand trends in global temperature, sea level rise, and carbon dioxide concentrations.

Key Objectives
  • Import and clean NASA climate datasets
  • Develop interactive visualizations using D3.js
  • Create user-friendly interface with filtering options
  • Implement machine learning for trend prediction
